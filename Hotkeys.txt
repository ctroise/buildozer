------------------------------------------
       ALT-SHIFT-CTRL hotkeys
------------------------------------------
ctrl-alt-v		 - extra line into a new variable
ctrl-left-mouse  - when done on a file's tab, will allow you to navigate to any part of its directory path
shft-shft        - bring up "Search anywhere" box (different from ctrl-shft-f search box)
ctrl-shft-s      - Settings
alt-i            - add to inline watches
alt-w            - add to watches
ctrl-F12         - navigate file structure (shows functions and such)
alt+F8	         - evaluate any selected expression while debugging
alt-enter        - Context menu (just for breakpoints?)	
alt-F7           - locate all occurrences of code referencing the symbol at the cursor
------------------------------------------------------------------------------------------
ctrl-o           - override method
ctrl-w			 - extend selection - do it a few times to select current method
ctrl-alt-F7      - show usages of a class
ctrl-shft-Bkspc  - go through last code edits in turn
ctrl-shft-i      - quick definition viewer
ctrl-p           - Inside method parens, bring up list of valid parameters
ctrl-shft-J		 - Join two lines together and remove unnecessary spaces
ctrl-shft-M		 - Match brace
ctrl-alt-e		 - highlight 'find' items only in current selection
ctrl-alt-t		 - "surround with" - context sensitive addition of "if", "while", "try/except" and so on
ctrl-shft-u		 - toggle case
ctrl-alt-shft-i  - Search inspections
Edit a language injection:  	Alt+Enter and choose Edit RegExp Fragment. The regular expression opens in a separate tab in the editor, where you can type backslashes as is.

    BOOKMARKS

shft-F11		Show all bookmarks
shft-F11-number - go to bookmark number 'number'
shft-F11-letter - go to bookmark mnemonic 'letter'
ctrl-F11		Create a "mnemonic" bookmark
shft-F11-g      Sudoku_GUI
shft-F11-c      config file
